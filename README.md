# Test Assignment

Это приложение — игра, в которой игроку необходимо, поворачивая трубы на плоской доске, довести сигнал из начальной точки до каждой конечной. Чем больше ходов потребуется игроку для завершения уровня, тем меньше очков он заработает. 

Каждый уровень состоит из:
* игровой доски `X * X` (зависит от сложности);
* `1` стартовой трубы, из которой исходит сигнал;
* X конечных труб с лампочками, ло которых нужно довести сигнал;
* `X*X-X-1` проемжуточных труб.

Этот проект — площадка для выполнения тестового задания. Он спроектирован в соответствии с подходами, принятыми в мини-играх Логопотам.

Проект завершен функционально: игровая логика в целом работает, визуальная составляющая достаточно информативна.

Ваши задачи:
1. Используя различные приемы и библиотеки, сделайте геймплей ярким и красочным. Добавьте аудио (фоновая музыка, отклик на действия), анимируйте переходы между состояниями, поддержите вибрацию при кликах или победе в уровне, шрифты — проявите фантазию, используя свои технические навыки. Можно использовать сторонние библиотеки для достижения красочных эффектов и экономии времени;
2. Добавьте на главный экран настройку звука: вкл/выкл, опционально слайдер громкости;
3. Исправьте неявные проблемы в коде;
4. Попробуйте исправить ошибку в игровой логике, если заметите ее.

**Рекомендуемое время: 2-3 часа.** Просим не тратить бОльшего времени, если не успеваете: для проблем, которые не успели исправить, опишите предлагаемые решения.

Мы не оцениваем качество используемых вами внешних материалов или красоту анимаций: не принципиально, если у аудиозаписи низкий битрейт или она плохо соответствует тематике игры; не принципиально, если какая-то анимация занимает странно много времени, и так далее. Нам необходимо оценить только техническое исполнение задачи.

Не ограничиваем вас в выборе библиотек, но для аудио желательно полагаться на `audioplayers`. Ожидаем уверенного использования `AnimationController`.

Для подбора аудио рекомендуем:
* воспользоваться заготовленными ассетами в папке `assets`;
* бесплатные ресурсы https://pixabay.com/music/
